# Reference — 通用參考資料

Tampermonkey 腳本的設計參考與建置文件。

## 分類

| 目錄 | 內容 | 適用 |
|------|------|------|
| `design-rules/` | 跨語言設計規則（命名、錯誤處理、檔案組織） | 所有語言 |
| `python/` | Python 編碼規則 + Server 設計 | Python 開發 |
| `tampermonkey/` | Tampermonkey 編碼規則 + 腳本設計 | 腳本開發 |
| `common/` | 通用 UI pattern（無限滾動、跳頁器、Popup 等） | 跨專案複用 |

## 使用方式

1. 新增腳本 → 先讀 `tampermonkey/INDEX.md`
2. 需要 UI pattern → 讀 `common/INDEX.md`
3. Python 開發 → 讀 `python/INDEX.md`
4. 設計決策 → 讀 `design-rules/INDEX.md`
